# Task Manager Web App

A full-stack Task Management Web Application that allows users to create projects, assign tasks, and track progress with role-based access control.

## 🚀 Features
- 🔐 User Authentication (JWT-based Signup/Login)
- 👥 Role-Based Access (Admin & Member)
- 📁 Project & Team Management
- 📋 Task Creation, Assignment & Status Tracking
- 📊 Dashboard with Task Insights (Pending, In Progress, Completed, Overdue)
- 🌐 Fully Deployed Backend

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Deployment: Railway

## 📌 Highlights
- RESTful API design
- Secure authentication & authorization
- Scalable backend architecture
- Real-world project structure
